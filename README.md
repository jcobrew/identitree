# identitree

reflective identity mapping app.

## what this repo is

this repository is a custom project by `jcobrew` / `jacobbsolana`.

it is built **on top of** the `make something` starter repository for tooling and scaffolding, but this repo itself is maintained as the `identitree` project and is not the original starter distribution.

## current direction

identitree is a game-like, pen-and-ink tree interface where:

- identity fruits connect to a central self node
- quests, activities, and skills live in the roots
- clicking nodes reveals reflective journal insights about identity growth
- a guided local chat-style intake maps user input into suggested identity pathways

## run locally

```bash
npm run dev
```

open `http://localhost:3000`.

## useful commands

- `$fixit` to run checks and fix common issues
- `$deploy` to publish a preview build to the internet
